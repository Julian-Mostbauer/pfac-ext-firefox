# Prime Factorizer

Prime Factorizer is a light Firefox extension that calculates and displays the prime factors of any number you encounter on the web. 

Whether you're a math enthusiast, student, or just curious, this tool makes finding prime factors as easy as a right-click.

## Features

* **Instant Factorization:** View the exact prime factorization of any number (e.g., `12 = 2^2 × 3`).
* **Color-Coded Highlights:** Numbers are highlighted for easy scanning. True primes get a green highlight, while composite numbers get a yellow highlight.
* **Factorize Whole Pages:** Scan and factorize every number on a webpage all at once.
* **Automatic Mode:** Enable automatic factorization in the extension settings to have numbers highlighted the moment a page finishes loading.
* **Smart Parsing:** Automatically ignores spaces, commas, and dots, seamlessly supporting both US and European number formats.
* **Handles Massive Numbers:** Built with modern web technologies to accurately factorize incredibly huge numbers (like `9,223,372,036,854,775,807`) without freezing or losing precision.
* **Blazing Fast:** Uses a smart in-memory caching system. If a number repeats on a page, it pulls from the cache instantly to keep your browsing experience smooth.

## How to Use

1. **Single Number:** Highlight any number on a webpage, right-click, and select **"Show Prime Factors"**.
2. **Whole Page:** Right-click anywhere on a page and select **"Factorize All Numbers"**.
3. **View Results:** Hover your mouse over any highlighted number to see its prime factors in a tooltip.
4. **Settings:** Click the extension icon in your toolbar or visit the Firefox Add-ons manager to enable "Automatic Mode".

## Privacy

Prime Factorizer operates **100% locally** within your browser. Zero data is collected, stored, or sent to any external servers.
